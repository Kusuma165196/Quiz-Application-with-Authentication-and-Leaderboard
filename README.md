# Quiz-Application-with-Authentication-and-Leaderboard
A full-stack Quiz Application with authentication and leaderboard features where users can sign up, log in, and take interactive quizzes. It includes randomized questions, instant feedback, and score tracking. User scores are stored and displayed on a dynamic leaderboard using HTML, CSS, JavaScript, Node.js, and JSON/MongoDB.
